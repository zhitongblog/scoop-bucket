# Ziplark Scoop bucket

Free, fast, cross-platform archiver — [ziplark.com](https://ziplark.com).

```powershell
scoop bucket add ziplark https://github.com/zhitongblog/scoop-bucket
scoop install ziplark
```

Installs the `ziplark` and `ziplark-mcp` command-line tools. For the desktop app,
see the [releases page](https://github.com/zhitongblog/ziplark/releases) or winget.
